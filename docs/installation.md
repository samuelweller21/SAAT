# Installation

## Library

```py
pip install saat
```

## Jupyter Demos

```
git clone https://github.com/samuelweller21/SAAT.git

# Intall [poetry](https://python-poetry.org/docs/#installing-with-the-official-installer)
poetry shell
poetry install

# Jupyter
python3 -m ipykernel install --user --name="SAAT" --display-name="SAAT"
```

## Run tests
```
pytest
```
